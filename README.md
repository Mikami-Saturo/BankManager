# BankManager

## Description
Application de gestion de comptes bancaires en Java.

## Classes
- **BankAccount** : classe principale avec les attributs account (numéro de compte) et balance (solde)
- **CheckingAccount** : compte courant, hérite de BankAccount, avec un attribut limit (limite)
- **SavingsAccount** : compte d'épargne, hérite de BankAccount
- **COD** : certificat de dépôt, hérite de BankAccount

## Comment exécuter
1. Ouvrir le projet dans IntelliJ IDEA
2. Exécuter la classe Main
